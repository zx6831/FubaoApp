## 宣传文案

让子女更安心，让长辈更容易坚持。福豹把每日健康任务、家庭关怀、健康记录和暖心话题连接在一起。

## 应用描述

福豹是一款为子女与长辈共同设计的家庭慢病管理和关怀提醒 App。

子女端可以创建健康计划、查看每日任务进度、关注健康记录与告警、复制暖心话术并管理家庭档案与设备；长辈端提供大字号、大点击区域和清晰朗读入口，帮助轻松完成用药、测量、运动和生活习惯任务。

主要功能：

- 手机验证码登录与一对子女、一位长辈的家庭绑定
- 血压、血糖、用药、运动、心情、体重与自定义计划
- 双端任务同步、周视图、月历和历史记录
- 健康数据记录、家庭火花、L1/L2 关怀提醒和健康周报
- 暖心话题、四类消息、模拟设备设置、数据导出和账号注销
- 断网缓存与联网后自动同步

福豹仅提供健康管理与关怀提醒，不提供诊断和治疗，不能替代医生的专业建议。

## 审核备注模板

应用包含“子女端”和“长辈端”，登录页可选择角色。审核环境需准备已绑定的两组账号和一个指定审核验证码；该验证码仅对服务端 `REVIEW_PHONE` 配置的手机号有效，不会通过 API 回传。

审核重点路径：子女登录 → 首页/计划/话题/我的；长辈登录 → 今日任务 → 完成任务 → 子女端进度同步。账号注销位于“子女端 → 我的 → 隐私与数据”，退出登录不会退出家庭组，长辈退出家庭组会返回邀请码页面。

设备为软件 V1 的模拟设备，不需要外部硬件即可完成发现、激活和设置流程。
